## 🚀 快速开始（推荐）

### 如果您是零基础用户：

1. **打开系统** → 登录管理员账号
2. **进入"自动申请脚本管理"页面**
   - 导航：管理 → 自动申请脚本管理
   - 或直接访问：`/admin/auto-apply-scripts`
3. **点击"创建新脚本"**
4. **填写表单**：
   - 选择学校
   - 输入申请URL
   - 选择是否需要登录
5. **点击"创建脚本"**
6. **完成！**

详细步骤请查看：[使用Web界面创建脚本-超简单指南.md](./使用Web界面创建脚本-超简单指南.md)

## ❓ 常见问题

### Q1: 我应该使用哪种方法？

**A**: 
- **零基础用户** → 使用方法一（Web界面）
- **有一定技术基础** → 使用方法二（脚本生成工具）
- **开发者** → 使用方法三（手动创建）

### Q2: 如何找到申请URL？

**A**: 
1. 访问学校的官方网站
2. 找到"申请"或"Apply"链接
3. 点击进入申请页面
4. 复制浏览器地址栏中的URL

### Q3: 如何测试脚本？

**A**: 
1. 设置环境变量：`PLAYWRIGHT_HEADLESS=false`
2. 启动开发服务器：`npm run dev`
3. 进入"可申请学校"页面
4. 点击"自动申请"按钮
5. 观察浏览器自动操作

### Q4: 字段无法自动匹配怎么办？

**A**: 
- 打开脚本文件
- 找到"字段映射"部分
- 添加手动映射规则
- 详细说明见各指南文档

### Q5: 脚本创建后在哪里？

**A**: 
- **文件位置**：`src/modules/auto-apply/schools/您的学校ID.ts`
- **注册位置**：`src/modules/auto-apply/autoApplyService.ts`
