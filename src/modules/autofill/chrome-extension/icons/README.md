# 图标资源说明

## 需要的图标尺寸

扩展需要以下尺寸的图标文件：

- `icon16.png` - 16x16 像素（浏览器工具栏）
- `icon32.png` - 32x32 像素（扩展管理页面）
- `icon48.png` - 48x48 像素（扩展详情页）
- `icon128.png` - 128x128 像素（Chrome Web Store）

## 图标设计建议

1. **主题**：学校申请、表单填充
2. **颜色**：可以使用渐变紫色（与 Popup 头部一致）
3. **元素**：可以包含：
   - 表单图标
   - 自动填充箭头
   - 学校相关元素

## 临时解决方案

在开发阶段，可以使用以下方法：

1. **使用在线图标生成器**：
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. **使用占位图标**：
   - 创建简单的纯色图标
   - 或使用系统默认图标

3. **使用图标库**：
   - Font Awesome
   - Material Icons
   - 转换为 PNG 格式

## 图标文件格式

- 格式：PNG（支持透明背景）
- 颜色模式：RGB
- 背景：透明或白色

## 注意事项

- 图标应该清晰可见，即使在 16x16 尺寸下
- 避免使用过多细节
- 保持品牌一致性
